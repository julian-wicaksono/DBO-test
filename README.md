
## Prerequisites
- Go v1.15 or higher
- PostgreSQL
- Gin-Gonic

## Installation

Clone the repository:

```bash
git clone https://github.com/tanmayVaish/go-auth
```

Go to the project directory:

```bash
cd go-auth
```

Download dependencies:

```bash
go mod tidy
```

Running the Project

```bash
go run main.go
```
